# Bundled Hyper-Extract Templates

This directory contains vendored Hyper-Extract preset YAML templates used by
`nexus.services.template_adapter.TemplateRegistry`.

The registry treats these files as template metadata and ontology guidance for
`knowledge_nexus`; it does not run Hyper-Extract's AutoGraph/AutoType runtime.
